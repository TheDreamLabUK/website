# Learning Objectives - Workshop 01 Morning: VS Code Setup

## Workshop Overview

Transform your computer into a professional AI command centre. By the end of this 3-hour session, you'll have a fully configured workspace that multiplies your productivity 10x.

## Primary Learning Outcomes

By the end of this workshop, you will be able to:

### 1. Installation & Configuration (Core Skills)

**You will be able to:**
- ✅ Install and configure VS Code on your operating system
- ✅ Navigate the VS Code interface confidently
- ✅ Install and manage extensions for your profession
- ✅ Configure AI assistants (Continue, Copilot, or Codeium)
- ✅ Set up development containers (optional but recommended)
- ✅ Customize settings for your workflow

**Success Criteria:**
- VS Code launches successfully
- At least 5 essential extensions installed
- AI assistant responds to prompts
- Custom settings configured

### 2. AI Integration Mastery

**You will be able to:**
- ✅ Connect multiple AI models to VS Code
- ✅ Write effective prompts for content generation
- ✅ Use AI for document enhancement and editing
- ✅ Understand cost vs. benefit of different AI providers
- ✅ Manage API keys and usage limits
- ✅ Switch between AI models based on task

**Success Criteria:**
- Successfully generated AI content
- Used at least 3 different prompt styles
- Can explain when to use different AI models
- Configured cost tracking

### 3. Productivity Automation

**You will be able to:**
- ✅ Create custom snippets for repetitive tasks
- ✅ Use keyboard shortcuts fluently (minimum 10)
- ✅ Leverage multi-cursor editing
- ✅ Set up project templates
- ✅ Automate file organization
- ✅ Build reusable workflows

**Success Criteria:**
- Created 3+ custom snippets
- Use keyboard shortcuts instead of mouse
- Organized project with template structure
- Automated at least one repetitive task

### 4. Project Organization

**You will be able to:**
- ✅ Create logical folder structures
- ✅ Use workspace configurations
- ✅ Implement file naming conventions
- ✅ Link related documents
- ✅ Manage multiple projects
- ✅ Export and share configurations

**Success Criteria:**
- Professional project structure created
- Workspace settings configured
- Files organized logically
- Can navigate project efficiently

### 5. Real-World Application

**You will be able to:**
- ✅ Complete a real project using new skills
- ✅ Apply AI to actual work tasks
- ✅ Measure productivity improvements
- ✅ Troubleshoot common issues
- ✅ Adapt workflow to your profession
- ✅ Share knowledge with colleagues

**Success Criteria:**
- Completed real project in under 30 minutes
- Documented time savings
- Can explain workflow to others
- Identified 3+ use cases for your work
